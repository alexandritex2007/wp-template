# wp−template

## Setup

### Clone folders
```
$ git clone https://github.com/alexandritex2007/wp-template.git
```
### Move to project directory
```
$ cd wp-template
```
### vagrant up
```
$ vagrant up
```
### site access

(wp-template.dev)


### Move to gulpfile directory
```
$ cd wp-template
```
### Installing packages
```
$ npm install
```
## Start-up
```
$ gulp
```
